Addition of two numbers


Input: num1 = 12, num2 = 5
Output: 17
Explanation: num1 is 12, num2 is 5, and their sum is 12 + 5 = 17, so 17 is returned. 

    
        public class Main {
            public static void main(String[] args) {
                int num1 = 12;
                int num2 = 5;
                int sum = num1 + num2;
                System.out.println("Sum: " + sum);
            }
        }
